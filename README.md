# Weather

## Welcome to my new project "Weather"!
## This is an application in which I trained my skills with using url, parsing json and using svg files. The data was obtained using the Yandex weather API

### Here were used UIKit, MVC
### This is a mini application for obtaining weather data in the city of interest.

#### In the first screen you will see the list of 10 cities and the weather in them.

![Screenshot1](https://github.com/Natalia-Sharapova/Weather/blob/main/MyWeather/Screenshots/Screenshot01.png?raw=true)

#### In the search bar you can find the city of interest

![Screenshot2](https://github.com/Natalia-Sharapova/Weather/blob/main/MyWeather/Screenshots/Screenshot02.png?raw=true)

#### By clicking on the name of the city, you will be taken to the next screen, where the weather will be indicated in detail
